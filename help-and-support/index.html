Using Data Export

Using Data Export is an easy way to get data extracted from your commecero account without having to interact with Comecero's powerful API.

App Settings

  • App Location - Realative URL path for this application. Default is /data-export
  • Company Name - Company Name that will appear at the top of the Data Export page. If blank, will use Company Name from your account settings.
  • Default Timezone - Allows you to change the default timezone for date times in the export. Default is UTC.

General Usage

Usage is pretty simple. Select the Data Set, Dates, and Export Format you would like to export. Other options to vary the export is "Include" subresources (Where available), Status (where available), and Export Format. Then simply click "Export."

Limitations

CSV exports are limited due to complex data structures returned in the API. Some attempt to "flatten" data results has been made, When exporting Orders and Invoices, each CSV row is actually at the item level instead of the order level. This means that order level data will be duplicated and specifically data that is numeric will not sum up correctly so be careful on how you use that data since it will not match the reporting tools within the comecero admin UI.

Code Availability

Code is available on github for you to customize, upload, and run in your accounts. If this tool does not suit your needs, or you will like to contribute, it is free to do so.
